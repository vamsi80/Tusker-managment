### 🎨 Issue 7: `react-doctor/design-no-redundant-size-axes` — `w-N h-N` → `size-N` ×112
Tailwind v3.4+ has a shorthand `size-N` that replaces matched dimensions like `w-5 h-5` with `size-5`. This makes styling incredibly clean and fast to read.

* **Sample Files:**
  * `src/app/w/[workspaceId]/p/page.tsx`
  * `src/components/task/kanban/kanban-card.tsx`
* **Fix:** Use VS Code Find & Replace (`Ctrl+Shift+H`) to map matching Tailwind dimensions (e.g. `w-4 h-4` ➔ `size-4`).
