## Complete DataTable Section Should Look Like:

```typescript
<DataTable
    columns={columns}
    data={flattenedData}
    searchKey="materialName"
    searchPlaceholder="Search materials..."
    filterFields={filterFields}
    filterDisplay="menu"
    rowSelection={rowSelection}
    onRowSelectionChange={setRowSelection}
    getRowId={(row) => row.id}
    enableRowSelection={(row) => !row.original.hasPO}
    getRowClassName={(row) => 
        row.original.hasPO ? 'opacity-50 bg-muted/30 cursor-not-allowed' : ''
    }
/>
```

## What This Does:

- **`opacity-50`**: Makes the row 50% transparent (dimmed)
- **`bg-muted/30`**: Adds a light gray background
- **`cursor-not-allowed`**: Shows ⛔ cursor when hovering
